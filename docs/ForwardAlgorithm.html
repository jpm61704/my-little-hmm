alphas' :: (Num p, Ord s, Foldable t) => (StateInfo s o p -> o -> Alpha p) -> HMM s o p t -> o -> AlphaMap s p Source #

the abstract calculation for the alphas of an observation o with respect to a model and observation. The first argument is a function that calculates the alpha value for a given state s_i given its StateInfo view and the observation. \ This is used to provide an abstraction so that addition data can be curried into the function before alpha's is called.
